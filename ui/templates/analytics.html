🔍 Methodology

  • Heatmap: The pitch is divided into 8 × 5 = 40 zones. Each event's (x, y) is derived deterministically from its minute, side, and text — home events cluster in the right half, away in the left. Zone colour scales with event count from blue (0) to red (max).
  • xG timeline: Per-event xG is approximated by type — goal = 0.75, save = 0.35, miss = 0.20, chance = 0.15, free_kick = 0.10, all others = 0. Lines are cumulative totals per side across 90 minutes.
  • Pass network: Player names are extracted from event text via regex; consecutive names in pass/assist/goal events form edges. Edge thickness scales with the number of connections. Nodes are sized by total involvement and laid out per side.
